<h1>PPTX to JPG Converter</h1>

<h5>This Repo Made for converting powerpoint files to images</h5>

Clone 
```bash
git clone https://github.com/miladkhajavi/convert-pptx.git
```

installing. 
```bash
pip install -r requirement.txt
```

Run
```bash
python main.py
```
